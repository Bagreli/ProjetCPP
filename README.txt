# instructions #

placer le fichier à la racine de votre ordi. 
Windows (sur lequel je travaille) : C:/


Sinon n'hésitez pas à modifier le chemin d'accès au fichier liste des mots. Le chemin relatif ne fonctionne pas à la compilation (merci CLion sur Windows?)


une variable de débugage "debug" est initialisée à 1 : cela permet de voir les logs (quel est le mot mystère choisi etc. Il vous suffit de la mettre à 0 pour que les différents messages disparaissent.


Pour le reste tout devrait fonctionner.
